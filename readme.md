# 图片转成营销号短视频脚本

### 环境准备
python 3.7+

### 使用方法
```text

git clone https://github.com/stars1324/pictureToVideo.git

pip install -r requirements.txt

文案写入content.txt

图片删除原本pictures的图片，换成营销号需要的图片

背景影音更换bgm里面，背景时长最好要超过视频时长

python main.py
```
